# Jarvice Waitlist

## Overview
A simple, elegant waitlist page for the Jarvice project built with Next.js and Tailwind CSS.

## Features
- Responsive design
- Email validation
- Waitlist submission handling

## Setup
1. Install dependencies:
   ```
   npm install
   ```

2. Run development server:
   ```
   npm run dev
   ```

3. Build for production:
   ```
   npm run build
   ```
